About Let's Go

Let's Go is a modern social network built with Flask and Neo4j graph database.

Features

  • User registration and authentication
  • Customizable user profiles
  • Follow/unfollow other users
  • Create and share posts with images
  • Like and comment on posts
  • Personalized feed from followed users
  • Explore and discover new users

Technology

This application leverages the power of graph databases through Neo4j, which is perfect for modeling social network relationships. Flask provides a lightweight and flexible web framework for the backend.

Open Source

This project is open source and available for learning and contribution.
